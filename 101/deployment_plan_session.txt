
Mistral has the following to interact with environments: 

[stack@undercloud 101]$ mistral --help | grep environment
  environment-create           Create new environment.
  environment-delete           Delete environment.
  environment-get              Show specific environment.
  environment-list             List all environments.
  environment-update           Update environment.
[stack@undercloud 101]$ 

A Mistral environment and Swift Container is created during deploy. 

Before deployment: 

stack@undercloud 101]$ swift list
overcloud-artifacts
[stack@undercloud 101]$

After starting a deployment (e.g. https://github.com/fultonj/oooq/blob/master/deploy-ceph-only.sh)

[stack@undercloud 101]$ swift list
overcloud
overcloud-artifacts
[stack@undercloud 101]$
